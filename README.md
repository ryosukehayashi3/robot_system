# robot_system
ロボットシステム学で利用したデバイスドライバ
***
# 実装内容  
講義で作成したデバイスドライバを使用し、  
echo 1 >> /dev/myled0  
上記のように入力するとLEDが点灯し  
echo 0 >> /dev/myled0  
と入力するとLEDが消灯する。  
***
# 必要なもの  
* Raspberry Pi 3+
* LED * 1  
* ブレットボード  
* 220Ω抵抗  
***

# ビルド方法  
`$ make`  
`$ sudo insmod myled.ko`  
`$ sudo chmod 666 /dev/myled0`
